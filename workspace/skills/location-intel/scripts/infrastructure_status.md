# Thai Infrastructure Status Reference
# Only projects marked FUNDED should count as upside in property analysis.
# PLANNED = zero value. PROPOSED = negative value (noise).

## Mass Transit (Updated 2026-04)

### OPERATIONAL (count in current value)
- BTS Sukhumvit Line: Mo Chit → Kheha (operational)
- BTS Silom Line: National Stadium → Bang Wa (operational)
- MRT Blue Line: Hua Lamphong → Lak Song (full loop operational)
- MRT Purple Line: Khlong Bang Phai → Tao Poon (operational)
- MRT Yellow Line: Lat Phrao → Samrong (operational)
- MRT Pink Line: Nonthaburi Civic Center → Min Buri (operational)
- Airport Rail Link: Phaya Thai → Suvarnabhumi (operational)
- Gold Line: Krung Thon Buri → Klong San (operational)

### FUNDED + UNDER CONSTRUCTION (count as upside)
- BTS Sukhumvit extension: Kheha → Samut Prakan (under construction)
- MRT Orange Line: Thailand Cultural Center → Min Buri (under construction, ~60% complete)
- MRT Purple Line south extension: Tao Poon → Rat Burana (under construction)
- MRT Red Line (Dark Red): Bang Sue → Hua Lamphong (under construction)

### PLANNED (do NOT count as upside)
- Chiang Mai Light Rail (Red/Blue/Green lines) — feasibility since 2010, no construction start
- Phuket Tramway — approved but no timeline
- BTS Samut Prakan further extension — mentioned in plans, no funding
- Any line not listed above with confirmed budget and construction contract

## Rule
When I say "future upside from [infrastructure]", I MUST verify its status in this file.
